# 俄罗斯方块 (Tetris Game)

一个使用Python和Pygame开发的经典俄罗斯方块游戏，支持完整的游戏功能和友好的用户界面。

## 🎮 游戏预览

![游戏预览](https://via.placeholder.com/800x600?text=俄罗斯方块+游戏预览)

## ✨ 功能特点

- 🎯 **完整的游戏体验**：实现了7种标准俄罗斯方块形状（I、J、L、O、S、T、Z）
- 🎮 **直观的操作控制**：左右移动、旋转、加速下落和硬下落功能
- 🏆 **得分系统**：根据消除行数、软下落和硬下落计算分数
- 📊 **关卡进阶**：随着分数增加，游戏速度逐渐提升
- 🎨 **精美界面**：清晰的游戏网格、方块渲染和操作指南显示
- 🎯 **用户友好**：开始界面、暂停功能和重新开始选项
- 📱 **可执行文件**：提供Windows可执行文件，无需安装Python环境

## 🛠️ 技术栈

- **编程语言**：Python 3.12+
- **图形库**：Pygame 2.6.1
- **打包工具**：PyInstaller 6.16.0

## 📋 安装说明

### 方法1：直接运行可执行文件（推荐Windows用户）

1. 下载 `dist/tetris.exe` 文件
2. 双击运行即可开始游戏
3. 无需安装Python或任何依赖

### 方法2：从源码运行

1. 确保已安装Python 3.12+
2. 安装Pygame库：
   ```bash
   pip install pygame
   ```
3. 运行游戏：
   ```bash
   python tetris.py
   ```

## 🎮 操作指南

| 按键 | 功能 |
|------|------|
| ← → | 左右移动方块 |
| ↑ | 旋转方块 |
| ↓ | 加速下落 |
| 空格键 | 硬下落（直接落到底部） |
| P | 暂停/继续游戏 |
| R | 游戏结束后重新开始 |
| 任意键 | 从开始界面进入游戏 |

## 📊 游戏机制

### 得分规则
- 消除1行：100 × 当前关卡分数
- 消除2行：300 × 当前关卡分数
- 消除3行：500 × 当前关卡分数
- 消除4行（俄罗斯方块）：800 × 当前关卡分数
- 软下落：每下落一格+1分
- 硬下落：额外奖励+2分

### 关卡进阶
- 每获得1000分提升一个关卡
- 关卡提升后游戏速度加快
- 初始下落速度：0.5秒/格
- 最大下落速度：0.05秒/格

## 🚀 构建可执行文件

如果您想自己构建可执行文件，请按照以下步骤操作：

1. 安装PyInstaller：
   ```bash
   pip install pyinstaller
   ```
2. 执行打包命令：
   ```bash
   python -m PyInstaller --onefile --windowed tetris.py
   ```
3. 可执行文件将生成在 `dist/` 目录中
